# Scroll Zoom Focus Effect

A scroll-based animation effect that creates a dramatic zoom and focus effect as the user scrolls.

## Features

- **Scroll-triggered animations** using CSS `animation-timeline: scroll(root)`
- **Starship zoom effect**: Starship scales up from 1x to 5x as you scroll
- **Scene unblur effect**: Background scene gradually unblurs and scales
- **Text animation**: Title text animates with letter spacing and opacity
- **Mix blend mode**: Text uses overlay blend mode for visual effect

## Files

- `scroll-zoom-effect.html` - HTML structure
- `scroll-zoom-effect.css` - CSS animations and styles

## Usage

1. Open `scroll-zoom-effect.html` in a browser
2. Scroll down to see the animation effects
3. The starship zooms in, the scene unblurs, and the text animates

Requires browsers that support:
- CSS `animation-timeline` (Chrome 115+, Edge 115+)
- CSS `animation-range`
- CSS `scroll()` function
